第29回_斥亲儿打醮动众人(3-0)
人物：旁白，凤姐，贾母，贾珍
旁白:话说贾母带领众人一行到清虚观打醮，到了门口，凤姐自己下了轿，想上来搀贾母下轿。可巧有个十二三岁的小道士儿，拿着剪筒，照管剪各处蜡花，正欲得便且藏出去，不想一头撞在凤姐儿怀里。凤姐便一扬手，照脸一下，把那小孩子打了一个筋斗，骂道。
凤姐：“野牛肏的，胡朝那里跑！”
旁白:那小道士也不顾拾烛剪，爬起来往外还要跑。众婆娘媳妇都喝声叫“拿，拿，拿！打，打，打！” 贾母听了忙问。
贾母：“是怎么了？”
旁白:贾珍忙出来问。凤姐上去搀住贾母，就回说。
凤姐：“一个小道士儿，剪灯花的，没躲出去，这会子混钻呢。”
旁白:贾母听说，忙道。
贾母：“快带了那孩子来，别唬着他。小门小户的孩子，都是娇生惯养的，那里见的这个势派。倘或唬着他，倒怪可怜见的，他老子娘岂不疼的慌？珍哥儿，你去好生带了来。旁白:贾珍只得去拉了那孩子来。那孩子还一手拿着蜡剪，跪在地下乱战。
贾母:拉他起来，孩子，你别怕。你几岁了？旁白:那孩子通说不出话来。贾母说道。
贾母:“可怜见的，珍哥儿，带他去罢。给他些钱买果子吃，别叫人难为了他。”
贾珍:是。
旁白:贾珍领了那个小道士出来，叫人来带去，给他几百钱，不要难为了他。 贾珍站在阶矶上，因问。
贾珍：“管家在那里？”
旁白:底下站的小厮们见问，都一齐喝声说：“叫管家！”登时林之孝一手整理着帽子跑了来，到贾珍跟前。贾珍道。
贾珍：“虽说这里地方大，今儿不承望来这么些人。你使的人，你就带了往你的那院里去；使不着的，打发到那院里去。把小幺儿们多挑几个在这二层门上同两边的角门上，伺候着要东西传话。你可知道不知道，今儿小姐奶奶们都出来，一个闲人也到不了这里。”
旁白:林之孝忙答应“晓得”，又说了几个“是”。贾珍道。
贾珍：“去罢。怎么不见蓉儿？”
旁白:一声未了，只见贾蓉从钟楼里跑了出来。贾珍道。
贾珍：“你瞧瞧他，我这里也还没敢说热，他倒乘凉去了！”
旁白:贾珍喝命家人啐他。那小厮们都知道贾珍素日的性子，违拗不得，有个小厮便上来向贾蓉脸上啐了一口。贾珍又道。
贾珍：“问着他！”
旁白:那小厮便问贾蓉道。
小厮：“爷还不怕热，哥儿怎么先乘凉去了？”
旁白:贾蓉垂着手，一声不敢说。贾珍又向贾蓉道。
贾珍：“你站着作什么？还不骑了马跑到家里，告诉你娘母子去！老太太同姑娘们都来了，叫他们快来伺候。”
旁白:贾蓉听说，忙跑了出来，一叠声要马，一面抱怨道。
旁白：“早都不知作什么的，这会子寻趁我。你们捆着手呢？马也拉不来。”
旁白:骂完待要打发小子去，又恐后来对出来，说不得亲自走一趟，骑马去了，不在话下。贾珍自去里面伺候，又领张道士进去见老太太。大家正看戏闲话，只见贾珍贾蓉的妻子婆媳两个来了，彼此见过，贾母方说。
贾母：“你们又来做什么，我不过没事来逛逛。”
旁白:一句话没说了，只见人报：“冯将军家有人来了。”原来冯紫英家听见贾府在庙里打醮，连忙预备了猪羊香烛茶银之类的东西送礼。凤姐儿听了，忙赶过正楼来，拍手笑道。
凤姐：“嗳呀！我就不防这个。只说咱们娘儿们来闲逛逛，人家只当咱们大摆斋坛的来送礼。都是老太太闹的。这又不得不预备赏封儿。”
旁白:刚说了，只见冯家的两个管家娘子上楼来了。接着赵侍郎也有礼来了。于是接二连三，都听见贾府打醮，女眷都在庙里，凡一应远亲近友、世家相与都来送礼。贾母才后悔起来说。
贾母：“又不是什么正经斋事，我们不过闲逛逛，就想不到这礼上，没的惊动了人。”旁白:因此虽看了一天戏，至下午便回来了，次日便懒怠去。凤姐又说。
凤姐：“打墙也是动土，已经惊动了人，今儿乐得还去逛逛。”
旁白:那贾母因昨日张道士提起宝玉说亲的事来，谁知宝玉一日心中不自在，回家来生气，嗔着张道士与他说了亲，口口声声说从今以后不再见张道士了，别人也并不知为什么缘故；二则林黛玉昨日回家又中了暑：因此二事，贾母便执意不去了。凤姐见不去，自己带了人去，也不在话下。
